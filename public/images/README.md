# Images Directory

This directory contains all images used throughout the HomeCanvas website.

## Directory Structure

- `hero/` - Hero banner background images
  - `hero-bg.jpg` - Main hero section background (recommended: 1920x1080px)

- `categories/` - Category tile images
  - `furniture.jpg` - Furniture category image (recommended: 800x800px, square)
  - `lighting.jpg` - Lighting category image (recommended: 800x800px, square)
  - `decor.jpg` - Decor category image (recommended: 800x800px, square)
  - `garden.jpg` - Garden category image (recommended: 800x800px, square)

- `collections/` - Collection carousel images
  - `[collection-slug].jpg` - Individual collection images (recommended: 1280x960px, 4:3 aspect ratio)
  - `default.jpg` - Fallback collection image

- `looks/` - Shop the Look images
  - `[look-slug].jpg` - Individual look images (recommended: 1280x960px, 4:3 aspect ratio)
  - `default.jpg` - Fallback look image

## Image Guidelines

- Use high-quality images (minimum 72 DPI for web)
- Optimize images for web (JPEG with quality 80-85% or WebP format)
- Maintain consistent aspect ratios within each category
- Images should reflect the HomeCanvas aesthetic: warm, natural, handcrafted furniture and home decor

## Image Sources

You can source images from:
- Unsplash (https://unsplash.com) - Search for: furniture, home decor, lighting, garden furniture
- Pexels (https://pexels.com) - Similar search terms
- Your own product photography
- Stock photo services

## Recommended Search Terms

- Furniture: "modern furniture", "wooden furniture", "scandinavian furniture", "living room furniture"
- Lighting: "pendant lights", "table lamps", "modern lighting", "interior lighting"
- Decor: "home decor", "interior design", "decorative accessories", "vases", "cushions"
- Garden: "outdoor furniture", "garden decor", "patio furniture", "planters"
